# Correspondence: As Above, So Below

> **TL;DR**: Patterns repeat across all scales of reality. Lane phase mechanics = teamfight mechanics = life mechanics. Master a pattern once, recognize it everywhere. This is how you become wise fast - you stop learning individual facts and start recognizing universal patterns.

## The Core Teaching

**Ancient Phrasing**: "As above, so below; as below, so above; as within, so without; as without, so within."

**What This Means**: The patterns that govern one level of reality also govern all other levels. What's true in your mind is reflected in your body. What's true in games is reflected in life. What's true at the micro scale is reflected at the macro scale.

**This isn't mysticism - it's pattern recognition.**

### Example: The Trading Pattern

**Lane Phase Trading:**
1. Identify enemy cooldown window (they used key ability)
2. Enter their range during weakness
3. Deal damage
4. Exit before they can respond
5. Repeat when next window opens

**Life Opportunity Trading:**
1. Identify opportunity window (situation where you have advantage)
2. Enter the situation
3. Extract value
4. Exit before disadvantage appears
5. Repeat when next opportunity appears

**Job Interview Trading:**
1. Identify company need (they're hiring for X skill)
2. Present yourself as solution
3. Demonstrate value
4. Close before they have doubts
5. Wait for next opportunity

**Same pattern. Different context. Once you see it, you see it EVERYWHERE.**

## Practical Applications

### In Gaming: Pattern Transfer

#### Pattern 1: Wave Management = Emotional Management

**Wave Management:**
- Push when you want pressure (aggressive)
- Freeze when you want safety (defensive)
- Reset when wave is out of control (neutral)

**Emotional Management:**
- Express when you need release (aggressive)
- Contain when you need stability (defensive)
- Reset when emotions are overwhelming (neutral)

**Same pattern**: Control → Express → Reset → Repeat

#### Pattern 2: Vision Control = Information Control

- Control vision → Control map

**Life:**

**Same pattern**: Build power → Use leverage → Extract value

## The Practice: Seeing Correspondences

### Week 1: Pattern Spotting
**Goal**: Notice one pattern appearing in multiple places

**Daily practice:**
1. Pick ONE pattern you understand well from gaming
   - Examples: Resource management, timing windows, positioning, information advantage
2. This week, look for that SAME pattern in 3 different areas:
   - School/work
   - Relationships
   - Personal habits
3. Journal: "Where did I see [pattern] today outside of gaming?"

**You'll discover**: The pattern you mastered in games exists EVERYWHERE

### Week 2: Pattern Application
**Goal**: Consciously apply gaming patterns to life

**Daily practice:**
1. Morning: "What gaming pattern can I apply today?"
   - Example: "Lane priority" → "Work priority"
2. Throughout day: Actively use that pattern
   - When I have lane priority (ahead on tasks), I can pressure other areas
3. Evening: "How did the pattern transfer?"

**You'll discover**: Gaming mastery = Life mastery, just different context

### Week 3: Pattern Creation
**Goal**: Start creating your own pattern library

**Daily practice:**
1. Create a "Pattern Library" document
2. List all patterns you've mastered:
   - Gaming patterns
   - Life patterns
   - Work patterns
3. Next to each, write: "Where else does this apply?"
4. Actively look for new applications

**You'll discover**: Your pattern library becomes a superpower - rapid learning in ANY domain
