## Technical SEO Implementation

### On-Page Optimization

**Title Tags** (55-60 chars):
- Format: `[Phenomenon]: [Key Insight] | Hermetic Academy`
- Example: `ADHD Explosion: Environment vs Biology | Hermetic Academy`

**Meta Descriptions** (150-160 chars):
- Hook + value prop + CTA
- Example: `Why is everyone being diagnosed with ADHD? Deep Hermetic analysis reveals vibrational mismatch between human biology and modern environment. Read the full decode.`

**URL Structure**:
```
/community/expert-decodes/[topic]-[key-insight]
Examples:
/community/expert-decodes/adhd-diagnosis-explosion-attention-crisis
/community/expert-decodes/modern-dating-apps-connection-crisis
```

**Heading Hierarchy**:
```
H1: Main title (only one per page)
H2: Major sections (principles, insights)
H3: Sub-sections within principles
```

**Image Optimization**:
- Alt text describing principle being illustrated
- File names with keywords (adhd-vibrational-mismatch.png)
- Compress for speed

**Internal Linking**:
- Link decodes to related decodes (cross-reference patterns)
- Link to principle explainers
- Link to wiki foundation articles

### Site-Wide SEO

**1. Homepage**
- Clear value prop: "Understand reality through Hermetic principles"
- Links to featured decodes
- Explanation of framework

**2. Decodes Hub Page** (`/community/expert-decodes/`)
- All decodes listed
- Filter by principle, topic, date
- Search functionality

**3. Principle Pages** (`/principles/[principle]`)
- Deep explanation of each principle
- Examples from multiple decodes
- Links to all decodes demonstrating principle

**4. Wiki Integration**
- Link decodes to relevant wiki articles
- Wiki articles link back to practical decode applications
